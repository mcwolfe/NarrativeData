# dataScience

A personal learning lab for simulation, modeling, and data analysis — spanning Python, R, and Julia.

## What's in here

**Python** — Agent-based modeling (Mesa), network analysis (NetworkX), discrete event simulation (SimPy), cellular automata, Bayesian analysis, machine learning basics, and various physics/math experiments.

**R** — Linear regression, network analysis, and exploratory workflows. Also used for teaching quantitative methods.

**Julia** — Early experiments with data exploration and visualization.

## About

This is a working portfolio of small experiments and learning projects, not polished production code. The common thread is using computation to understand systems — whether that's trade networks, rumor spreading, cellular worlds, or superhero statistics.

Marvel character data (FASERIP.csv) is derived from the classic 1984 Marvel Super Heroes tabletop RPG and is used here for educational purposes only.
